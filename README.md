# Fintech-LLM-Data-Pipeline-Insights
POC de pipeline de dados com PostgreSQL, Python, LangChain e Ollama para gerar insights textuais automatizados a partir de transações de uma fintech, com consulta SQL, processamento analítico e exportação dos resultados em CSV.
